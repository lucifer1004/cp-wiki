## Day07

- 暴力。枚举 $[\min,\max]$ 范围内的每个整数，找到代价函数的最小值。
- 更好的方法
  - 第一问：可以利用绝对值函数性质得到在中位数处总代价最小；
  - 第二问：每一段上的代价函数是一个二次函数，可以利用二次函数性质求出这一段上的最小值；从左到右依次处理，从一段移动到另一段时，可以在 $\mathcal{O}(1)$ 时间内更新二次函数的系数。
